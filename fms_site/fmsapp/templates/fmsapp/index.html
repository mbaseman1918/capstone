Welcome to the Fire Management System

Current application features:

  • Allow a user to create an account which includes both a physical address and an e-mail address
  • Takes the physical address of the user and geocodes (translates to latitude and longitude) it thru the google maps API
  • Acquires fire incident location information from Fire Information for Resource Management System (FIRMS)
  • Displays user's physical address and fire incident location, allowing near-real time (NRT) comparison
  • Displays user's physical address and a heatmap of fire incidents allowing an estimate of the intensity of nearby fire incidents

Proposed future application features:

  • Create a manager user group with incident managemant features
  • Create a fire crew user group with crew management features
  • Allow e-mail alerts to be sent out
  • Allow predictive modeling of fire spread
  • Incorporate a weather API to get information on past weather and forecast of weather
  • Predict time estimates for digging fireline
