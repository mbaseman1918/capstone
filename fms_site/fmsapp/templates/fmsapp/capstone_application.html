Under continual development

As part of the program at PDX Code Guild I have begun working on a capstone project. The goal of the project is to develop an application which can alert users of wildfires in near real-time. In order to develop this capability data from Fire Information for Resource Management System (FIRMS) was used. The goal of the application is to allow a user to enter a specific area and obtain current relevant informaiton on whether or not a fire is occuring nearby.
