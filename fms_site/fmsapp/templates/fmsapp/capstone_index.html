Phoenix

Currently under development

As part of the program at PDX Code Guild I have begun working on a capstone project. The goal of the project is to develop an application which can alert users of wildfires in near real-time. In order to develop this capability data from the National Aeronautics and Space Administration's (NASA) Earth Observing System Data and Infromation System(EOSDIS) was used. The goal of the application is to allow a user to enter a specific area and obtain current relevant informaiton on whether or not a fire is occuring nearby.

Phoenix
Project Overview Wildfires are one of the major forms of natural disasters which regularly occur in the Pacific Northwest. Early detection is one of the best ways to mitigate the amount of damage from a wildfire. The intent of Phoenix Fire is to create an application that will allow near real time detection of Wildfires.
